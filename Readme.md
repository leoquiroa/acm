**Set up** 
Create a file called `keys.json` in the root of the project

	{
		"OPENAI_API_KEY":"...",
		"OUTSCRAPER_API_KEY":"..."
	}

**Installation**

    cd ACM
    python -m venv venv
    .\venv\Scripts activate
    pip install openai
    pip install outscraper

**Use**
The file `Prompt.txt` has the instruction to ChatGPT
