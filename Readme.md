To set up create a file called keys.json in the root of the project
    {
        "OPENAI_API_KEY":"...",
        "OUTSCRAPER_API_KEY":"..."
    }

Installation
    cd ACM
    python -m venv venv
    .\venv\Scripts activate
    pip install openai
    pip install outscraper
Use
    Prompt.txt has the instruction to ChatGPT
